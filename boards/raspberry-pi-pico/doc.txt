## Flash the Board

To flash the board you must have the **Picotool** available in your system (also added to $PATH).
For Picotool source and documentation check [here.](https://github.com/raspberrypi/picotool)

Additionally, to be able to flash the board using Picotool and without root privileges, add the
following udev rule:

```ATTR{idVendor}=="2e8a", ATTRS{idProduct}=="0003", MODE="660", GROUP="plugdev", TAG+="uaccess"```

to, */etc/udev/rules.d/80-pico.rules*

Now, to flash the board, hold the *BOOTSEL* button on the board while powering the board.

After that the following command should work:

```make BOARD=raspberry-pi-pico flash```
