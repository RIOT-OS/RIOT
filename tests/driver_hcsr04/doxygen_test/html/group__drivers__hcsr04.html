◆ hcsr04_temp_to_sound_speed()

uint16_t hcsr04_temp_to_sound_speed ( uint16_t  temp)

Function that calculates the speed of sound.

Parameters
[in,out]tempTemperature in mili Celsious

It uses the following formula: v = s * sqrt(1 + T/To) Where:

  • s is the speed of sound at 0 Celsious or 273 Kelvin
  • T is the given air temperature
  • To is the reference temperature 273 Kelvin
Warning
Temperature must be given in milli Celsious
